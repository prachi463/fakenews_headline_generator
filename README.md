# 📰 Fake News Headline Generator (Python)
## 🚀 Overview

The **Fake News Headline Generator** is a Python-based project that generates random and humorous "Breaking News" headlines using predefined datasets.

This project demonstrates the use of randomization, basic logic building, and user interaction in Python.

---

## ✨ Features

- 🎲 Random headline generation  
- 🔁 Multiple outputs using loop  
- ⌨️ User input to control execution  
- 🧠 Simple and clean logic  
- ⚡ Lightweight and fast  

---

## 🛠️ Tech Stack

- **Language:** Python  
- **Module Used:** `random`
